Basic Aggregate C++ Data Types
::::::::::::::::::::::::::::::

Just like Python comes with basic data structures that are composed of atomic datatypes like Lists and Tuples,
C++ has ways to aggregate atomic items, namely arrays and `structs` (structures / classes). An array contains items of identical type
and structures contain different types.
